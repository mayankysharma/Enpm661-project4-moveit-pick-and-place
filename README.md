# Enpm661-project4-moveit-pick-and-place

## Run instructions:- 
Open terminal
```
cd ws_moveit/
catkin build
source ~/ws_moveit/devel/setup.bash
roslaunch panda_moveit_config demo.launch
```
Open new terminal
```
cd ws_moveit/
catkin build
source ~/ws_moveit/devel/setup.bash
rosrun moveit_tutorials pick_place_custom

```
